# mauiverse.net
Powering the mauiverse.net website.

## Quick start

```sh
npm install
npm run dev
```

Starts the site locally at `http://localhost:4321`.

## Common npm scripts

- `npm run dev` - runs the site locally and executes pre-dev generation/fetch tasks.
- `npm run create:author -- <githubUsername>` - creates a new author YAML file.
- `npm run create:feed-item -- <title-or-url> [date] [githubUsername]` - creates a new community feed markdown file.
