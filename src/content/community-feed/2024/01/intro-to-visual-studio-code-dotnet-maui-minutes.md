This episode is a quick orientation to the early MAUI-in-VS Code experience and what the extension unlocks for cross-platform development. Maddy keeps it grounded in setup and day-one workflow so you can decide whether the editor path fits how you like to work.

## What you'll learn

- What the .NET MAUI extension for VS Code brings to the workflow.
- How getting started differs less than you might expect across Windows, Mac, and Linux.
- Why VS Code can be an appealing option for developers who want a lighter MAUI environment.
