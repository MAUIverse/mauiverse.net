Session by Jose Javier Columbie Did you know you can now host Blazor components in .NET MAUI apps to build cross-platform native apps? In this introductory session we will cover everything you need to get your app running everywhere. Yes, MAUI Blazor apps can run in Windows, Mac, iOS, and Android. If you like Blazor you will love MAUI.

### What you will learn

- Core concept and where it fits in a .NET MAUI app
- Step-by-step setup and implementation flow
- Platform considerations and practical trade-offs
- How to apply the pattern in real projects

### Good to know

- The demo focuses on practical usage, so validate package/version details against your current MAUI target before shipping.
