No scripts, no safety net — just three .NET MAUI engineers, an AI agent, and a real app taking shape in real time. 

Will they do it?

This live stream showcases MauiDevFlow, an open-source toolkit from `dotnet/maui-labs` that enables AI agents to inspect, build, and debug .NET MAUI apps through visual tree inspection, live property editing, and terminal-driven screenshots.

## What you'll learn

- How to set up MauiDevFlow in a .NET MAUI project from scratch
- How AI agents build and modify UI through code and live inspection
- Real-time debugging with visual tree inspection, CDP (Chrome DevTools Protocol), and application logs
- The full build → deploy → inspect → fix feedback loop in action
- How MauiDevFlow integrates with tools like GitHub Copilot for AI-assisted development
