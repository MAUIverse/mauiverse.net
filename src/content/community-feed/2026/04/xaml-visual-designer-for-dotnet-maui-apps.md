XAML development in .NET MAUI can be verbose and trial-and-error heavy without a visual surface. Hector walks through a XAML Visual Designer tool that aims to make building .NET MAUI UIs more visual and interactive.

## What you'll learn

- How a visual designer surface integrates with .NET MAUI XAML development
- How visual tooling can reduce the feedback loop when building cross-platform UI
- Practical demonstrations of designing MAUI interfaces visually
