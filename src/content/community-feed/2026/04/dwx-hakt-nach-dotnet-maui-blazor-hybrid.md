In this DWX (Developer World) session, Codrina explores .NET MAUI Blazor Hybrid — the approach that lets web developers leverage their existing Blazor and web skills to build native mobile and desktop apps with offline capability and full platform access.

## What you'll learn

- How .NET MAUI Blazor Hybrid bridges the gap between web development and native mobile apps
- How to reuse Blazor web application components in a .NET MAUI context
- The benefits of a single codebase for both web and native platforms
