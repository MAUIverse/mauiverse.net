Syncfusion followed up their first open-source Toolkit release with a second batch of controls targeted at common app UI needs.

This set includes **Bottom Sheet**, **Button**, **Calendar**, **Numeric Entry**, and **Numeric UpDown**, alongside **.NET 9** support and a round of Toolkit improvements and bug fixes.
