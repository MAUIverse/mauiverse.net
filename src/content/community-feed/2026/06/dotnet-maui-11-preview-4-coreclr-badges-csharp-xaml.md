Gerald Versluis covers everything new in .NET MAUI 11 Preview 4 in this video walkthrough. This is a preview release where the team is actively seeking early community feedback.

## What you'll learn

- CoreCLR is now the default runtime for .NET MAUI on Android, iOS, and Mac Catalyst
- Material 3 design support landing for Android
- `dotnet watch` now works on mobile for faster iteration
- Badge support for Shell tabs and ToolbarItems
- Writing C# code directly in XAML expressions
- Compiled bindings in DataTemplates
- Monochrome icon support for Android adaptive icons
- How to provide early feedback to the .NET MAUI team on preview features
