Version 4.4.127 of Grial UI Kit is a substantial release that brings .NET 10 support while retiring .NET 8, along with major changes to navigation and popup infrastructure. The navigation bar handler has been fully reworked for better performance and cross-platform consistency, and Mopups has been replaced with UXDivers' own popup framework.

## What you'll learn

- What changed with .NET 10 support and why .NET 8 was retired
- How the rebuilt `GrialNavigationPageHandler` improves performance and eliminates platform inconsistencies
- Why UXDivers built their own popup engine to replace Mopups, and how to migrate
- The four new popup templates: Clipboard Link, Single Entry Form, Interactive Notification, and Options List
- How to add notification badges to toolbar items using the new `ToolbarItemBadgeProperties`
- Repeater control improvements: measure strategy, buffer size, and native empty view support
- Breaking changes and migration steps for upgrading
