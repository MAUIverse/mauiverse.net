If you're familiar with Bootstrap's design system, this project applies that same styling to stock .NET MAUI controls. Get a clean, consistent look across platforms using CSS theming — no custom renderers or handlers required.

## What you'll find

- Bootstrap CSS theming applied to built-in .NET MAUI controls
- Familiar Bootstrap design system for cross-platform apps
- CSS-based styling without custom renderers or handlers
