When Copilot can run the app, see the UI, and read logs and screenshots without constant supervision, iteration speed stops being linear. David documents the results of that shift — eight significant .NET MAUI projects shipped in just two weeks.

## What you'll learn

- **MAUI Sherpa:** A Blazor Hybrid desktop app for managing your entire MAUI dev environment — SDKs, devices, emulators, certs, and provisioning profiles
- **MauiDevFlow:** Agentic dev loop assists that give AI agents eyes and hands inside a running MAUI app
- **PolyPilot / .NET MAUI Terminal / .NET MAUI Linux:** Experimental platform targets expanding MAUI's reach
- **.NET MAUI macOS and tvOS:** Native AppKit and tvOS backends for .NET MAUI
- **.NET MAUI Bootstrap Themes and Skills:** CSS theming and AI coding assistant skills for MAUI development
- How AI-assisted development changes the feedback loop for building developer tools
