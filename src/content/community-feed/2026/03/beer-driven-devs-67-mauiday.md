Fresh off .NET MAUI Day London, Matt and Liam settle in for a wide-ranging debrief that covers the event highlights, community atmosphere, and some inventive side projects.

### What you will learn

- Key takeaways from .NET MAUI Day London, including talks by David Ortinau and Gerald Versluis
- Matt's "Opinionated by Necessity" talk on Singleton vs Transient page lifetimes and common MVVM misconceptions
- MotiMe — an IoT physio app that turns exercises into a game for kids, with 92%+ engagement improvement
- Stephane's talk on C# expressions in XAML and what it means for the future of MAUI UI development
- The coffee app demo using on-device AI and a phone camera to count a room
- Matt's "Fluff Slicer" — a browser extension that evaluates YouTube video quality via transcript analysis
- Liam's Encubed Games — a fully agentic AI game studio built on Claude Code and Cloudflare
- Candid Q&A insights on .NET MAUI's future at Microsoft and what .NET 10 delivers
