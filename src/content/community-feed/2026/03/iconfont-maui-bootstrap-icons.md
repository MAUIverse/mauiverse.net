Icon fonts are one of the cleanest ways to add scalable, resolution-independent icons to your .NET MAUI app. This package wraps the full Bootstrap Icons set — 2,078 icons — with strongly-typed C# constants so you can reference them by name.

## What you'll find

- The complete Bootstrap Icons set packaged as a .NET MAUI icon font
- Strongly-typed glyph constants for referencing icons in XAML and C#
- Drop-in installation with no raw Unicode values to manage
