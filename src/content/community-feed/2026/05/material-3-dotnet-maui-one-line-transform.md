Material 3 (Material You) brings Google's modern design language to .NET MAUI on Android, and Gerald Versluis shows just how easy it is to enable. One project file property is all it takes to transform the look and feel of your app.

## What you'll learn

- What Material 3 (Material You) is and how it differs from previous Material Design versions
- Enabling Material 3 on your .NET MAUI project with a single property
- The current state of Material 3 support for iOS and other platforms
- Exploring the HealthProfile sample app that demonstrates Material 3 controls in action
- Visual before-and-after comparisons of .NET MAUI controls with Material 3 enabled
- Using the .NET MAUI Sample Browser to explore Material 3 examples
