Syncfusion announced their first set of open-source .NET MAUI controls as part of the **Syncfusion Toolkit for .NET MAUI**.

This post provides the overview: what’s included (Charts, Carousel, Tab View, Segmented Control, Chips, Effects View, Shimmer, Pull-to-Refresh, Text Input Layout, Navigation Drawer, etc.), where to get it (**NuGet + GitHub**), and how they’re encouraging community participation and contributions.
