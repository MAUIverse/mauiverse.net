You probably know .NET MAUI, you know Blazor Hybrid, but did you know you can still get platform native controls, but don't have to write XAML? Instead, use Blazor syntax, no XAML! That is what you can do with the .NET MAUI Blazor Bindings and the BlazorBindings.Maui package, which we will see in this video.

### What you will learn

- .NET MAUI Blazor Bindings
- Three Ways to use .NET MAUI
- What is the Blazor Bindings?
- Install BlazorBindings.Maui Templates
