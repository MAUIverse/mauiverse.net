With MauiReactor we have another great option to do MVU and coded user-interfaces for .NET MAUI. With this framework you can leverage the MVU pattern to build component-based UIs built on top of .NET MAUI to realize amazing cross-platforms apps. Join me in this video to get started with MauiReactor!

### What you will learn

- MauiReactor: MVU for .NET MAUI
- Ways to create .NET MAUI apps without XAML
- MauiReactor GitHub Repository
- Install Templates & Hot Reload

### Good to know

- The demo focuses on practical usage, so validate package/version details against your current MAUI target before shipping.
