<p>In this episode of the Xamarin Podcast, Pierce Boggan is joined by James Montemagno to discuss what is new for Xamarin developers, including new features in the latest Visual Studio for Mac preview, Xamarin.Forms 2.3.4/2.3.5-pre, and updates to Visual Studio Mobile Center.</p>

<p>Show Notes: </p>

<p>5:00 - What&#39;s new in Visual Studio for Mac (C# 7, web editors, Azure publishing, .NET Core enhancements)</p>

<p>10:30 - Xamarin documentation is now multilingual (with Spanish, Portuguese, and Chinese localizations)</p>

<p>12:00 - Updates to Xamarin University, including monthly pricing</p>

<p>14:00 - Xamarin.Forms 2.3.4</p>

<p>14:55 - Device.OS &amp; TargetPlatform changes in Xamarin.Forms 2.3.4</p>

<p>19:15 - Xamarin.Forms 2.3.5 prerelease</p>

<p>22:15 - Bringing macOS to Xamarin.Forms</p>

<p>25:15 - TLS 1.2 overview</p>

<p>28:15 - Updates to Visual Studio Mobile Center</p>

<p>32:00 - Developer Tool of the Week</p>
