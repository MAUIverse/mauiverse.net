<p>In this episode of the Xamarin Podcast, James Montemagno and Pierce Boggan announce some exciting changes surrounding the Xamarin Podcast and cover what&#39;s new in Xamarin, including the stable release of Cycle 9, the upcoming Visual Studio 2017 launch, and more.</p>

<p>Show Notes:</p>

<p>0:00:30 - Exciting changes coming to the Xamarin Podcast</p>

<p>0:04:45 - Visual Studio 2017 launch</p>

<p>0:06:30 - The new Visual Studio 2017 workload-based installer</p>

<p>0:11:00 - The Xamarin release process</p>

<p>0:18:30 - New &quot;bootstrapped&quot; cross-platform templates</p>

<p>0:22:20 - New item templates for Xamarin.Forms</p>

<p>0:25:00 - Xamarin.Forms IntelliSense, Info.plist Editor, and various other Visual Studio enhancements</p>

<p>0:27:15 - Updates to Xamarin Studio, including .NET Standard support, multiprocess debugging, and more</p>

<p>0:33:30 - Running Visual Studio for Mac and Xamarin Studio side-by-side</p>
