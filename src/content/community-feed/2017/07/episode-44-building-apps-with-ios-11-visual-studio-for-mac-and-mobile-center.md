<p>In this episode of the Xamarin Podcast, James and Pierce discuss the Xamarin iOS 11 web preview, the latest in Visual Studio for Mac, and our experiences using Visual Studio Mobile Center.</p>

<p>3:50 - XAML productivity tools in Visual Studio for Mac (MFractor)<br>
6:10 - Mastering the Android Support Libraries<br>
9:00 - How to install Visual Studio 2017 offline - great for conferences and trainings<br>
11:15 - Updates to Mobile Center<br>
13:30 - Apps built with Xamarin - Urban Refuge, Guitar Center, Nintendo Switch<br>
19:20 - New Xamarin previews, including our first iOS 11 web preview<br>
25:00 - What&#39;s new in Visual Studio for Mac<br>
29:25 - Package/Tool of the Podcast</p>
