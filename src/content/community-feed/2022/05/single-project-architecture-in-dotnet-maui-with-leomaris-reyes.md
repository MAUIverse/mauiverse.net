In this episode of Snorkeling in MAUI, Leomaris Reyes walks through .NET MAUI's single project architecture and what it means for cross-platform development. She explains how developers can share assets across platforms while still having the freedom to handle platform-specific needs when required.

### What you will learn

- How .NET MAUI's single project structure works compared to the older multi-project Xamarin approach
- How to leverage shared assets like fonts, images, and splash screens across all platforms
- When and how to use platform-specific code within the single project model
- Why this architecture gives developers more flexibility without sacrificing consistency
