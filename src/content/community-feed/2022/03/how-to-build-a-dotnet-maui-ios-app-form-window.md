This tutorial demonstrates the remote build workflow for developing a .NET MAUI iOS app from Windows by pairing with a Mac.

## What you'll learn

- How Windows-to-Mac pairing enables iOS build and debug for MAUI.
- What prerequisites are needed on both machines before building.
- How to validate that remote toolchain integration is working.

Note: iOS builds still require Apple tooling on a Mac, even when coding from Windows.
