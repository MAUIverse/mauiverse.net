Want to learn the basics of git? Check out this tutorial! In this video I will walk you through a brief history of version control systems and then go on talking about what git is and how to use it. While I'm talking the commands will pop up on screen, like: git push, git pull, git commit, git add, etc. Additionally you'll learn about forks, clones, branches and more! After watching this, you will know how to get started with git yourself!
