In this month's episode we reboot the podcast! Matt Soucoup, a Senior Cloud Advocate at Microsoft joins James as host and we expand the topics to focus both on Xamarin mobile development and using Azure with your mobile apps.

We also cover the latest in Azure news, including DevOps, MSAL, and new Azure Cosmos DB pricing.
