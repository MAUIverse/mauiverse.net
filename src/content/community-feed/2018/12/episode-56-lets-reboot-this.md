<p>In this month&#39;s episode we reboot the podcast! Matt Soucoup, a Senior Cloud Advocate at Microsoft joins James as host and we expand the topics to focus both on Xamarin mobile development and using Azure with your mobile apps.</p>

<p>In this episode we recap the latest news in the Xamarin world including a marathon Twitch workshop, wondering Android build time improvements, the latest news in Xamarin.Forms, and recent announcements from Microsoft Connect();. We also cover the latest in Azure news, including DevOps, MSAL, and new Azure Cosmos DB pricing.</p>

<p>All of this, and a whole lot more!</p>
