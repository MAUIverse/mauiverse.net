## Get Started

For development on Windows, install the latest .NET MAUI workload with [Visual Studio 17.12 Preview 1](https://visualstudio.microsoft.com/vs/preview/).

For development on Windows or Mac, install the [.NET MAUI extension](https://marketplace.visualstudio.com/items?itemName=ms-dotnettools.dotnet-maui) in VS Code and install the latest workload from the [standalone installer](https://dotnet.microsoft.com/en-us/download/dotnet/9.0) using the `dotnet workload install maui` command.
