## Summary

The focus for .NET MAUI in .NET 8 is SDK quality, as evidenced by the work listed below. Also new in this release, we have split the packaging of .NET MAUI between NuGets and Workloads [#11206](https://github.com/dotnet/maui/pull/11206). This will enable more flexibility when switching between versions of .NET MAUI via NuGet packages on top of the base workload install. There are no changes to how you install .NET MAUI as a result of this. We will elaborate on this in future previews of .NET 8.

* [Announcing .NET 8 Preview 1](https://devblogs.microsoft.com/dotnet/announcing-dotnet-8-preview-1/)
* [Installation instructions](https://github.com/dotnet/core/blob/main/release-notes/8.0/preview/8.0.0-preview.1.md#net-multi-platform-app-ui-maui-workload-downloads)
* [Known Issues](https://github.com/dotnet/maui/wiki/Known-Issues/)
* [Pinning to older version of .NET with global.json](https://learn.microsoft.com/dotnet/core/tools/global-json)
* [.NET uninstall tool](https://learn.microsoft.com/en-us/dotnet/core/additional-tools/uninstall-tool)

### Compatibility

This release is NOT fully compatible with Visual Studio. To use this preview of .NET MAUI expect to use CLI to install, build, and run.

Recommended platform versions to build against:

* Xcode 14.2 which includes iOS, iPadOS, and Mac Catalyst 16.2
* Android API 33
* Windows App SDK 1.2.221109.1
