## What's Changed

* Add IProvideParentValues as required service for AppThemeResourceExtension by [Gerald Versluis](/community-contributors/jfversluis/) in https://github.com/CommunityToolkit/Maui/pull/2460
* Add UnconditionalSuppressMessage to fix IL2092 by [Gerald Versluis](/community-contributors/jfversluis/) in https://github.com/CommunityToolkit/Maui/pull/2461
* Fix XamlRoot being set incorrectly in Popup Handler by [James Crutchley](/community-contributors/ne0rrmatrix/) in https://github.com/CommunityToolkit/Maui/pull/2476
* Fix statusBar color changes on modal pages by [Pedro Jesus](/community-contributors/pictos/) in https://github.com/CommunityToolkit/Maui/pull/2413
* Fix Snackbar layout #1901 by [Vladislav Antonyuk](/community-contributors/VladislavAntonyuk/) in https://github.com/CommunityToolkit/Maui/pull/2456
* Fix options not working for Core by [James Crutchley](/community-contributors/ne0rrmatrix/) in https://github.com/CommunityToolkit/Maui/pull/2491
* Enhance DrawingView and DrawingViewService to provide the ability to export the full image size by [Shaun Lawrence](/community-contributors/bijington/) in https://github.com/CommunityToolkit/Maui/pull/2193
* Enable Expander on ListView + Enable Expander CollectionView GridLayout 1557 by [Vladislav Antonyuk](/community-contributors/VladislavAntonyuk/) in https://github.com/CommunityToolkit/Maui/pull/1567
* Fix XamlC warning for `DrawingViewModel` and MVVMTK warning `SpeechToTextPage` by [James Crutchley](/community-contributors/ne0rrmatrix/) in https://github.com/CommunityToolkit/Maui/pull/2513
