When using the MediaElement on Android the volume bar would keep popping up whenever you interacted with the MediaElement control. This version fixes that! 

Additionally, the `MediaWidth` and `MediaHeight` properties weren't populated, not cool! So as of this version, those are working again too 👏 

## What's Changed

* fixed sample application crashed with MediaElement for Windows by [Oleksii Sokol ](/community-contributors/olekssokol/) in https://github.com/CommunityToolkit/Maui/pull/1870
* Update Media Element to fix code compliance by [James Crutchley](/community-contributors/ne0rrmatrix/) in https://github.com/CommunityToolkit/Maui/pull/1955
* Prevent volume bar from popping up on Android by [Gerald Versluis](/community-contributors/jfversluis/) in https://github.com/CommunityToolkit/Maui/pull/1965
* Reinstate MediaElement MediaWidth & MediaHeight functionality by [Gerald Versluis](/community-contributors/jfversluis/) in https://github.com/CommunityToolkit/Maui/pull/1950
