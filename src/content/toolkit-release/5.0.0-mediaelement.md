## Breaking Changes

**Major Breaking Changes**
- .NET 8 no longer supported
- Xcode 16.1 Required 
- Increase minimum supported OS versions of `CommunityToolkit.Maui.MediaElement`
  - Android 26.0
  - iOS 15.0
  - MacCatalyst 15.0

## What's Changed
* Support .NET 9.0 by [Shaun Lawrence](/community-contributors/bijington/) in https://github.com/CommunityToolkit/Maui/pull/2215
*  [[BUG] .Net 9 Android MediaElement stops playing video ](https://github.com/CommunityToolkit/Maui/issues/2373)
