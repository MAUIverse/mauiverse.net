Happy holidays for those of you who celebrate! In the meanwhile, we're not clowning around and present to you the last release of 2023.

This version has a good number of bugfixes and improvements, but nothing too crazy so that it might blow up your project before you go on a holiday break. As always a big thank you to anyone and everyone not only using our product but also caring, engaging and even contributing. We appreciate it!

Let's make 2024 even better 🚀 

## What's Changed

* Add support for setting badge count using `UNUserNotificationCenter` … by [Vladislav Antonyuk](/community-contributors/VladislavAntonyuk/) in https://github.com/CommunityToolkit/Maui/pull/1531
* Use fully qualified namespace for Colors reference in TextToColorGenerator by [@IIARROWS](/repo-contributors/community-toolkit/?q=IIARROWS) in https://github.com/CommunityToolkit/Maui/pull/1536
* Remove SkiaSharp.Views Dependency by [@JoonghyunCho](/repo-contributors/community-toolkit/?q=JoonghyunCho) in https://github.com/CommunityToolkit/Maui/pull/1547
* Fix IOS/MacOS AvataView background color with image bug by [@TRybina132](/repo-contributors/community-toolkit/?q=TRybina132) in https://github.com/CommunityToolkit/Maui/pull/1539
* [BUG] [Windows] UniformItemsLayout item positioning not correct #1553 by [Vladislav Antonyuk](/community-contributors/VladislavAntonyuk/) in https://github.com/CommunityToolkit/Maui/pull/1566
* Add sample to trigger an animation using the AnimateCommand by [Shaun Lawrence](/community-contributors/bijington/) in https://github.com/CommunityToolkit/Maui/pull/1577
* Removes the transparent color set on Android when null by [Victor Hugo Garcia](/community-contributors/vhugogarcia/) in https://github.com/CommunityToolkit/Maui/pull/1584
