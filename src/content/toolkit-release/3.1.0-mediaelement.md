Years in the making but here it is! Fullscreen support for our MediaElement 🤯  A big shoutout to [James Crutchley](/community-contributors/ne0rrmatrix/) and [Victor Hugo Garcia](/community-contributors/vhugogarcia/) but also [Pedro Jesus](/community-contributors/pictos/) [Shaun Lawrence](/community-contributors/bijington/) and [@brminnick](/repo-contributors/community-toolkit/?q=brminnick) for all their work, reviews and probably some blood, sweat, tears, the occasional fights with partners, sacrifices to the binary father and who knows what else.

Also as part of the fullscreen change you should now be able to use MediaElement when using Shell, not using Shell, inside of a CollectionView, etc. Also something many people have been waiting for.

Please note that we did update the [MediaElement docs](https://learn.microsoft.com/dotnet/communitytoolkit/maui/views/mediaelement?tabs=android#platform-specific-initialization) with a note about using fullscreen on Android, so be sure to read that.

Go grab this release and try it out in your apps!

## What's Changed

* Fix IOS bug in media Element when using Popup by [James Crutchley](/community-contributors/ne0rrmatrix/) in https://github.com/CommunityToolkit/Maui/pull/1597
* Media Element Full Screen Support for Android and Windows by [Victor Hugo Garcia](/community-contributors/vhugogarcia/) in https://github.com/CommunityToolkit/Maui/pull/1692
* Fixes #1702: Fixed Memory Leak in MediaElement by [@marco-skizza](/repo-contributors/community-toolkit/?q=marco-skizza) in https://github.com/CommunityToolkit/Maui/pull/1736
