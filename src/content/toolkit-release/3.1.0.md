Whoops! Look like our smart little source generators were causing slow IntelliSense in your .NET MAUI apps. We've fixed that and also added some other improvements to this new and shiny v3.1.0 release. And yes, we're again releasing this on a Friday 😌

## What's Changed
* Use `[GeneratedRegex]` by [@brminnick](/repo-contributors/community-toolkit/?q=brminnick) in https://github.com/CommunityToolkit/Maui/pull/763
* [BUG] DrawingView GetImageStream cuts off lines on edge of image #773 by [Vladislav Antonyuk](/community-contributors/VladislavAntonyuk/) in https://github.com/CommunityToolkit/Maui/pull/788
* fix: Removed wrong attributes on ExpanderPage. by [@HavenDV](/repo-contributors/community-toolkit/?q=HavenDV) in https://github.com/CommunityToolkit/Maui/pull/790
* Include link and brief description to the projects board in the readme by [Shaun Lawrence](/community-contributors/bijington/) in https://github.com/CommunityToolkit/Maui/pull/731
* IsInRangeConverter by [@GeorgeLeithead](/repo-contributors/community-toolkit/?q=GeorgeLeithead) in https://github.com/CommunityToolkit/Maui/pull/582
* Update ToastDuration options documentation with actual duration by [Gerald Versluis](/community-contributors/jfversluis/) in https://github.com/CommunityToolkit/Maui/pull/793
* Make `TextColorTo` Generated Classes for .NET MAUI Controls Internal by [@brminnick](/repo-contributors/community-toolkit/?q=brminnick) in https://github.com/CommunityToolkit/Maui/pull/791
* Fix the Incremental behavior of our Source Generator by [Pedro Jesus](/community-contributors/pictos/) in https://github.com/CommunityToolkit/Maui/pull/815
* Bumped a bunch of dependencies
