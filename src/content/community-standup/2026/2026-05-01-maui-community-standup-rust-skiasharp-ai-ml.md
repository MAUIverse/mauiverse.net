This Community Standup brings together some of the most exciting experimental work happening in the .NET MAUI ecosystem. Nick Kovalsky (taublast) demonstrates integrating Rust code into .NET MAUI apps, using SkiaSharp for cross-platform rendering, and performing real-time AI/ML processing on live camera feeds — all running inside a .NET MAUI application.

David Ortinau and Gerald Versluis host as Nick walks through practical demos of each topic, showing how these technologies combine to unlock new capabilities for .NET MAUI developers.
